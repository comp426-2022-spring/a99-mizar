## API Documentation

 
 Use sentiment anaylsis (rudimentary for now). Go to this link in your browser:
 ```bash
localhost:<port>/sentiment/tweet=<tweet-text>
 ```
 For example:
 ```bash
localhost:3000/sentiment/tweet=Nothing can take me away from my love of cheese
 ```
